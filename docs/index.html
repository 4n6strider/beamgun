Beamgun is a tiny Windows program that stops rogue USB devices like:

Rubber Duckies

Malicious keyboard devices that deliver arbitrary payloads on target.

LAN Turtles

Network adapters that can hijack connections and steal credentials.

Mass Storage

Exfiltrate large volumes of data quickly and locally from a target.

Beamgun runs in the background, quietly listening for USB devices.

Beamgun is open source (AGPLv3) and totally free (as in beer).
